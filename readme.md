# Nano Realms ⚔️

A multiplayer online 'MUD' like game using microservices and graph database

Tech stack:

- Backend API and core engine in Go
- Websockets and REST
- Datastore is a Neo4j graph database
- Frontend written in Vue.js and TypeScript
